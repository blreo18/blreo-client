블레오 접속기 / 단축키 / 기본 사용 팁 안내

1. 접속 및 실행 방법
- 접속기는 관리자 권한으로 실행해 주세요.
- 기존 구버전 문구(0.5.3 이상 / 0.5.4 통합)는 오래된 안내입니다.
- 현재는 새로운 클라이언트 파일 기준으로 접속해 주세요.
- 가급적 최신 공유 버전(예: Blreo_0.5.8.4.zip)으로 사용해 주세요.
- 버전이 맞지 않으면 추후 접속이 제한될 수 있습니다.
- 다중 접속 시, 실행 후 원하는 클라이언트 수를 고려해서 접속 버튼을 누르고, 로그인 창이 뜨면 다시 접속 버튼을 눌러 접속합니다.
- 창모드 사용 시 접속기에서 창모드를 체크해 주세요.
- 패치 또는 업데이트가 있었을 경우 접속기를 다시 실행한 뒤 접속해 주세요.

2. 키보드 조작 및 기본 단축키
- 접속기에서 키보드 입력 / 키보드 조작을 켜면 방향키와 스페이스바로 사냥할 수 있습니다.

[기본 조작]
- 방향키 : 캐릭터 이동
- 스페이스바 : 몬스터 선택
- 숫자키 2 ~ 4 : 선택한 대상 공격 / 스킬 사용
- Alt : 떨어진 아이템 이름 보기
- F5 : 자동줍기 / 보이지 않는 드랍템 확인용으로 자주 사용
- W : 달리기
- I : 아이템창
- C : 스탯창
- S : 스킬창
- F1 : 정보 확인

[다중 클라이언트 조작]
- 창모드 다중 실행 시 Alt + Tab으로 전환하면서 플레이하면 편합니다.

3. 자동물약 설정
- 접속기에서 자동물약을 체크합니다.
- 원하는 사용 퍼센티지를 입력합니다.
- 게임 내 슬롯은 아래처럼 맞춰 주세요.
  1번 슬롯 : 체력 물약
  2번 슬롯 : 마나 물약
  3번 슬롯 : 스태미나 물약

[추가 팁]
- 로그인 시 자동물약 숫자 입력이 방해될 경우, 자동물약 퍼센트를 0으로 두고 먼저 로그인한 뒤 다시 30% / 50% 등 원하는 수치로 변경해 주세요.

4. 편의 기능 설정
- 상점 이용 시 세금이 적용될 경우 접속기에서 세금 항목을 체크해 주세요.
- 드랍 아이템이 보이지 않을 때는 F5를 눌러 확인해 보세요.
- Alt 키로 떨어진 아이템 이름을 먼저 확인해 보세요.
- 랭킹페이지 > 드롭인포에서 드랍 정보를 확인할 수 있습니다.

5. 채팅 및 기본 명령어
- ! 입력 후 말하면 전체 채팅입니다.
- 파티 채팅은 /파티 할말 형식으로 사용합니다.

[초기 지원금]
- IP당 1회 사용 가능하며, 아래 명령어 입력 시 각 10만 골드가 지급됩니다.
- !님아돈좀
- !제발돈좀

6. 유료 버프(레블)
[일반 버프]
- !공업
- !방업
- !경치
- !드랍
→ 5분 / 3만원 / 10% 상승

[강화 버프]
- !강공업
- !강방업
- !강경치
- !강드랍
→ 5분 / 5만원 / 15% 상승

[사용 예시]
- !공업 10
→ 50분 버프 적용 / 비용 50만원

[참고]
- 초보 구간에서는 공방업에 무리해서 레블을 쓰기보다, 레벨업과 기본 파밍에 집중하는 편이 효율적일 수 있습니다.
- 경치는 65까지 지원된다는 안내가 있었으며, 초보는 공방업보다 드랍 위주가 낫다는 의견도 있었습니다.

7. 채널 시스템
- 일부 사냥터에는 채널 시스템이 적용되어 있습니다.
- !채널 입력 시 채팅창에 채널 존재 여부, 채널 수, 접속 인원 등의 정보가 출력됩니다.

[채널 확인 및 이동]
- !채널 : 채널 정보 확인
- !채널 1
- !채널 2
- !채널 3
- !채널 4
- !채널 5

[참고]
- 채널 수는 운영 상황에 따라 변경될 수 있습니다.
- 대화상 언더루인 1층, 시드왕 3층, 보석광산 등 일부 지역의 채널 확장 / 증설 언급이 있었습니다.

8. 이동 / 특수 기능
[레블머신 / 도박장]
- !나락
→ 레블머신 / 도박장 관련 명령어

[경주장 이동]
- !나락2
→ 경주장 즉시 이동

[소환 / 출두]
- !소환 캐릭명
  예: !소환 홍길동
- !출두 캐릭명
  사용 방식은 소환과 동일합니다.

[다중 소환]
- !소환 창조 섹시
- 위와 같이 여러 이름을 띄어쓰기로 입력하면 여러 명 소환이 가능합니다.
- !출두도 동일한 방식으로 사용할 수 있습니다.

9. 초보 시작 가이드
- 처음 시작 위치는 초보자던전입니다.
- 초반에는 토끼부터 잡으면 됩니다.
- 사냥터나 이동 위치를 모를 경우 F1을 눌러 정보 확인을 해보세요.
- 보석광산 등 일부 지역은 F1 또는 맵 정보를 통해 위치를 확인할 수 있습니다.
- 베르니안 마을에서 맵을 켜고 보석광산 방향을 확인해 이동하라는 유저 안내도 있었습니다.

10. 기타 팁
- 다중 클라이언트 플레이 시 창모드 + Alt + Tab + 키보드 조작 조합이 가장 편합니다.
- 드랍이 안 보이면 Alt, F5를 먼저 확인해 보세요.
- 접속 시 인벤토리에서 장착 아이템 1개를 한번 해제 후 다시 장착하면 튕김이 줄어드는 경우가 있습니다.
  예: 신발, 벨트, 반지 등 쉬운 장비 1개
- 엘프 / 하엘의 경우 화살은 10발당 1개 소비되도록 수정되었습니다.
- 화살 카운트 수량은 11시 방향 아이디 옆에 표시됩니다.
- 파티 추가 경험치는 다른 IP의 파티원일 때만 적용됩니다.
- 같은 IP의 다중 클라이언트는 파티 추가 경험치가 적용되지 않을 수 있습니다.
- 파티 걸이 거리는 20으로 증가되었다는 운영자 언급이 있었습니다.
- 레블 소지 최대치는 21억이라는 운영자 언급이 있었습니다.

11. 경제 / 수수료 / 운영 규칙
- 레블 거래 시 거래 중인 레블에서 3% 수수료가 차감됩니다.
- 강화는 4강부터 시도 시 10만 레블 수수료가 발생합니다.
- 레블이 부족한 상태에서 강화 시도 시 아이템은 장비 그대로 실패 처리될 수 있습니다.
- 경주장 당첨금에는 10% 수수료가 차감됩니다.

12. 트러블슈팅
[V3 / 백신 관련]
- 접속기 파일이 삭제되거나 실행이 막히면 V3 실시간 감시를 잠시 해제해 주세요.
- 이후 블레오 폴더를 검사 예외 폴더로 등록해 주세요.

[접속 꼬임 / 실행 문제]
- 접속이 안 되거나 파일이 꼬였을 경우 기존 폴더를 정리한 뒤 새 파일을 다시 압축 해제해 주세요.
- 새 폴더에 다시 설치한 뒤 실행해 보세요.
- 인스턴스 폴더가 정상 생성되는지 확인해 주세요.
- 접속이 안 될 경우 접속기와 클라이언트를 모두 종료 후 다시 실행해 보세요.
- 업데이트가 있었을 경우 접속기를 다시 켜서 업데이트 후 접속해 주세요.

[다중 클라이언트 문제]
- 다중 클라이언트가 더 이상 켜지지 않거나 7번째 이상에서 종료되면 인스턴스 폴더를 모두 삭제 후 다시 실행해 보세요.
- 기존 클라 인스턴스 삭제 후 접속해야 해결되는 경우가 있습니다.

13. 버그 및 제보 사항
- 조합 후 재접속 시 원재료 상태로 돌아가는 현상이 발생할 수 있습니다.
- 해당 현상이 발생하면 바로 운영자에게 제보해 주세요.
- 채널 이동 후 일부 상황에서 몬스터 피가 닳지 않거나 접속 이상이 발생했다는 제보가 있었습니다.
- 시드왕 방 입장 시 팅김 제보가 있었습니다.
- 일부 지역(예: 언더풋 계열로 보이는 지역)에서는 물약이 안 먹는다는 유저 제보가 있었습니다.
- 패치 직후 강화창 / 업데이트 중 임시 장애가 발생할 수 있습니다.
- 그 외 문제점들은 공지 댓글 또는 운영자에게 제보해 주세요.

14. 참고 / 유저 팁 모음
- 초반에는 버프보다 선레벨업 후 템파밍이 편하다는 의견이 있었습니다.
- 드랍이 안 나오면 드롭테이블을 먼저 확인해 보세요.
- 60존 판데보다 자얀 쪽이 더 낫다는 유저 팁이 있었습니다.
- 레블머신(!나락), 축비, 메너 / 이칲 판매, 작드 이후 파밍 등은 유저들 사이에서 언급된 레블 수급 팁입니다.
- 보석광산, 언더루인, 시드왕 3층, 작드, 자얀, 나괴 등은 유저들 사이에서 자주 언급되는 주요 사냥 / 파밍 구간입니다.

15. 마무리
- 신규 유입은 기존 공유 파일이나 서랍 / 공지 접근이 안 보일 수 있으니, 필요 시 접속기와 공지 파일을 다시 받아 주세요.
- 안내 내용은 운영 / 패치 상황에 따라 수시로 변경될 수 있습니다.
- 최신 변경사항은 채팅방 공지와 운영자 안내를 우선 확인해 주세요.
- 그 외 문제점들은 이곳 공지에 댓글로 남겨주세요.
