1. 신규 유저 빠른 시작
============================================================

[처음 해야 할 것]
1. 최신 운영자 공유 ZIP 파일을 받습니다.
2. 접속기를 관리자 권한으로 실행합니다.
3. 로그인 화면에서 사용할 아이디 / 비밀번호를 직접 입력해 접속합니다.
4. 아이디와 비밀번호는 반드시 따로 기록해 둡니다.
5. 게임 내 F1 안내를 먼저 확인합니다.
6. 카카오톡 닉네임은 대표 캐릭터명 또는 주요 캐릭터명으로 맞춰 두는 것을 권장합니다.

[초기 지원금]
- !님아돈좀 : IP당 1회, 10만 골드 지급
- !제발돈좀 : IP당 1회, 10만 골드 지급
- 두 명령어를 각각 입력하면 총 20만 골드를 받을 수 있습니다.
- 같은 공유기 / 동일 IP 환경에서는 일부만 적용될 수 있습니다.

[초반 사냥 흐름]
- 초보사냥터
- 버그던전 / 왕벌
- 미노사냥터 / 미노
- 작은드래곤
- 자얀
- 나괴
- 레드

[초보 팁]
- 처음 시작 위치는 초보자던전입니다.
- 초반에는 토끼부터 잡으면 됩니다.
- 사냥터나 이동 위치를 모를 경우 F1을 확인해 주세요.
- 보석광산 등 일부 지역은 F1 또는 맵 정보를 통해 위치를 확인할 수 있습니다.
- 초반에는 공방업에 무리해서 레블을 쓰기보다 레벨업과 기본 파밍에 집중하는 편이 효율적일 수 있습니다.
- 드랍 아이템이 보이지 않으면 Alt로 이름을 확인하고 F5를 눌러 보세요.
- 펫은 필수는 아니지만 자동줍기와 버프 때문에 초반부터 소환을 추천합니다.
- !아이템뽑기 3은 하루 3회 무료 뽑기 기준으로 안내되었습니다.

2. 설치 / 접속 / 계정 / 다중 실행
============================================================

[기본 실행]
- 접속기는 관리자 권한으로 실행해 주세요.
- 신규 유입은 공지만으로 클라이언트 파일 위치가 바로 보이지 않을 수 있으니, 최신 운영자 공유 ZIP 파일을 먼저 받아 주세요.
- 접속기는 자동업데이트 방식입니다.
- 업데이트가 반영되지 않으면 접속기를 완전히 종료한 뒤 다시 실행해 주세요.
- 접속이 안 되거나 파일이 꼬였을 경우, 기존 폴더를 정리한 뒤 최신 압축본을 새 폴더에 다시 풀어서 실행해 주세요.

[계정 생성]
- 별도 회원가입 없이 접속기 로그인 화면에서 사용할 아이디 / 비밀번호를 입력해 접속하는 방식입니다.
- 아이디와 비밀번호는 반드시 따로 기록해 두세요.
- 캐릭터명은 소환 / 출두 명령어 사용 편의를 위해 너무 길지 않게 만드는 것을 추천합니다.

[다중 접속]
- 실행 후 원하는 클라이언트 수를 고려해서 접속 버튼을 눌러 로그인 창을 띄운 뒤, 다시 접속 버튼을 눌러 접속하면 됩니다.
- 창모드 / 키보드 입력 / 세금 패치는 현재 기본 사용 기준입니다.
- 다중 클라이언트가 더 이상 켜지지 않거나 특정 번째 클라이언트부터 종료되는 경우, 모든 클라이언트를 종료한 뒤 instance 폴더를 삭제하고 접속기를 다시 실행해 보세요.
- 인스턴스가 꼬인 경우에는 새 폴더에 다시 압축 해제 후 실행하는 것이 가장 빠릅니다.

[다중 조작 팁]
- 창모드 다중 실행 시 Alt + Tab 또는 작업표시줄 클릭으로 클라이언트를 전환하면서 플레이하면 편합니다.
- 작업표시줄에 클라이언트 순서를 고정해 두면 Windows + 숫자키로 창 전환이 가능합니다.
- 창이 많아질수록 실수 입력이 늘 수 있으므로, 클라이언트 위치를 고정해 두는 것을 추천합니다.
- 다클라 자체는 허용 흐름이지만, 본인이 직접 컨트롤 가능한 범위에서 운용하는 것을 권장합니다.
- 자동사냥 기능 개발 예정은 없습니다.

3. 기본 조작 / 단축키
============================================================

[기본 조작]
- 방향키 : 캐릭터 이동
- ` : 몬스터 선택 / 타겟 선택
- 숫자키 1~3 : 물약 슬롯
  - 1번 슬롯 : 체력 물약
  - 2번 슬롯 : 마나 물약
  - 3번 슬롯 : 스태미나 물약
- 숫자키 4~9 : 선택한 대상 공격 / 스킬 사용
- 숫자키 0 : 스킬 또는 추가 단축 슬롯으로 활용 가능
- Alt : 떨어진 아이템 이름 보기
- F5 : 자동줍기 / 보이지 않는 드랍템 확인용
- W : 달리기
- I : 아이템창
- C : 스탯창
- S : 스킬창
- F1 : 정보 확인 / 안내 확인

[몬스터 선택키]
- 기존 안내에 스페이스바 몬스터 선택으로 남아 있는 내용이 있을 수 있으나, 현재 기준은 ` 키입니다.
- ` 키는 일반 키보드에서 숫자 1 왼쪽, Tab 위쪽의 물결표 키 위치입니다.
- 노트북 키보드에서는 배열에 따라 위치가 다를 수 있으니 숫자 1 왼쪽 키를 먼저 확인해 주세요.
- 채팅 중 ` 키를 누르면 몬스터 선택 / 타겟 변경이 될 수 있으니 주의해 주세요.

[권장 전투 입력]
- 1~3번 슬롯은 물약 기준입니다.
- 공격 / 스킬은 4번부터 등록하는 것을 권장합니다.
- 마우스 조작이 불편할 경우, 키보드 이동 + ` 몬스터 선택 + 숫자 4~9 공격 조합을 사용해 보세요.
- 반복 사냥 시 각 클라이언트에서 ` → 4 같은 방식으로 타겟 선택 후 공격하는 방식이 많이 쓰입니다.

4. 자동물약 / 자동줍기 / 드랍 확인
============================================================

[자동물약 설정]
- 접속기에서 자동물약 사용 퍼센트를 입력합니다.
- 게임 내 슬롯은 아래처럼 맞춰 주세요.
  - 1번 슬롯 : 체력 물약
  - 2번 슬롯 : 마나 물약
  - 3번 슬롯 : 스태미나 물약

[자동물약 주의]
- 로그인 시 자동물약 숫자 입력이 방해될 경우, 자동물약 퍼센트를 0으로 두고 먼저 로그인한 뒤 다시 30% / 50% 등 원하는 수치로 변경해 주세요.
- 자동물약은 즉시 무적 기능이 아닙니다. 현재 체력보다 큰 피해가 한 번에 들어오면 사망할 수 있습니다.
- 일부 지역에서는 자동물약 또는 물약 사용이 정상 작동하지 않을 수 있다는 제보가 있으므로 확인이 필요합니다.

[드랍 확인]
- 드랍 아이템이 보이지 않을 때는 Alt로 이름을 확인하고 F5를 눌러 확인해 보세요.
- 펫을 소환하면 자동줍기와 펫 버프가 적용됩니다.
- 단, 펫은 필수는 아니며 필요에 따라 소환 / 해제할 수 있습니다.

[드랍 소유권 / 사라짐]
- 드랍 직후에는 해당 몬스터를 잡은 캐릭터 기준으로 획득됩니다.
- [유저 제보] 약 1분 후에는 주인 관계없이 다른 캐릭터도 획득 가능한 것으로 알려져 있습니다.
- [유저 제보] 약 2분 후에는 바닥 드랍 아이템이 사라질 수 있습니다.
- 중요한 드랍은 Alt / F5 / 펫줍기 여부를 바로 확인해 주세요.
- 정확한 시간은 운영 상황에 따라 변경될 수 있으므로 최신 공지를 우선 확인해 주세요.

6. 펫 시스템
============================================================

- 파티 걸이 거리는 20으로 증가되었습니다.

- 접속이 안 될 경우 접속기와 클라이언트를 모두 종료 후 다시 실행해 보세요.
